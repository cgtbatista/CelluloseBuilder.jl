CelluloseBuilder.PBC — Method
PBC(xsize::Int64, ysize::Int64, zsize::Int64; phase="Iβ", pbc=nothing)

Returns (ncells, lattice) where ncells is the number of unit cells along x, y and z axes and lattice is the unit cell vector of this cell. This function is used to define the periodic boundary conditions required to build the cellulose crystal.
